# VOOHU RJ45 Connector – Technical Resources

This repository contains engineering resources for **VOOHU RJ45 connectors**, including:

- Datasheets (PDF)
- PCB footprints & 3D models
- Application notes (Industrial Ethernet, PoE, EMI)
- Selection guides for RJ45 with magnetics

## Key Product Lines

- **RJ45 Connector with Magnetics (MagJack)** – 10/100/1000Base-T, PoE/PoE+
- **Industrial RJ45** – Shielded, waterproof (IP67), wide temperature
- **SFP Connector & Cage** – For high-speed networking
